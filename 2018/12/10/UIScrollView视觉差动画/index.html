UIScrollView视觉差动画

iOS
动画
发布日期:   2018-12-10

引言

在一些app中或者一些博客动画系列里面,经常可以看到UIScrollView视觉差动画效果.整体的效果比UIScrollView的左右切换更为优雅.用户体验可以赚取一些分数.下面来对这种效果的实现做一些具体的分析.

预览

  1. 视觉差动画效果

  1. 普通的切换效果

分析

  1. UIScrollView切换显示不同的图片,原理是给UIScrollView添加不同的UIImageView,通过UIScrollView来切换展示不同的图片;

  2. 视觉差效果,其实就是图片位置的偏移.图片显示则用到UIImageView.而为了改变图片位置,则需要给UIImgeView包裹一层父视图.所以我们这里用到了AnimationView来作为UIImageView的父视图.

  3. 当UIScrollView向左滑动的时候,当前正在显示区域的AnimationView向左移动,为了保持AnimationView上面的图片可以不随着AnimationView同步滑动,我们需要设置AnimationView的UIImageView的横坐标大于0.相反当前正在滑动的AnimationView后面的一个AnimationView的UIImageView的横坐标小于0;

  4. 当前显示在屏幕上的AnimationView,我们可以通过UIScrollView的偏移量来计算得到其对应的index.当有手势左右滑动的时候可以得到其前一个或者后一个index对应的AnimationView和AnimationView下的UIImageView;

  5. 图片需要移动的距离我们可以给定一个具体的数值,在手势左右滑动的时候,我们修改AnimationView对应的UIImageView的横坐标,在滑动过程中我们不断去修改这个UIImageView的横坐标,当滑动结束的时候,我们再去复位这个UIImageView的横坐标.

  6. 用一个公式来简明表示就是: UIImageView.x = 需要移动的距离 - 移动的比例*需要移动的距离;

  7. 移动的比例: 

    // 就是UIImageView在AnimationView的偏移量除以UIScrollView的宽度

leftRation = ( UIScrollView.contentOffset.X  - (index)*UIScrollView.width)/UIScrollView.width;

rightRation = (UIScrollView.contentOffset.X  - (index+1)*UIScrollView.width)/UIScrollView.width;

代码

- (void)scrollViewDidScroll:(UIScrollView *)scrollView {

    NSInteger index = scrollView.contentOffset.x/scrollView.bounds.size.width;
    CGFloat contentOffsetX = scrollView.contentOffset.x;

    AnimationView *leftAnimationView  = [scrollView viewWithTag:(index+100)];
    AnimationView *rightAnimationView = [scrollView viewWithTag:(index+1+100)];

    // AnimationOffset 定义的一个宏,定义为100,可以自己修改
    CGFloat scrollViewWidth = scrollView.bounds.size.width;
    CGFloat moveDistance = scrollViewWidth - AnimationOffset;
    CGFloat leftRation   = (scrollViewWidth*(index+1) - contentOffsetX)/scrollViewWidth;
    CGFloat rightRation  = (contentOffsetX - scrollViewWidth*index)/scrollViewWidth;

    leftAnimationView.imageViewX  = moveDistance - moveDistance*leftRation;
    rightAnimationView.imageViewX = -moveDistance + moveDistance*rightRation;
}
- (void)setImageViewX:(CGFloat)imageViewX {
    _imageViewX = imageViewX;
    self.imageView.frame = CGRectMake(imageViewX, 0, self.imageView.bounds.size.width, self.imageView.bounds.size.height);
}
